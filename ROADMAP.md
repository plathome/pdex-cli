# Roadmap

## PDEX : A CLI tool for PDexchange environments

Pdex is a great tool for Pdexchange development environments.

This tool comprises of the following flexibilities,i.e.,

- easy installation in windows, linux and mac
- scriptability
- scalability
- simple design
- simple interfacing
- stable
